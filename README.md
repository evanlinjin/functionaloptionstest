# Functional Options Test
A test repo for learning how to set up functional options for Golang.

## Usage
Installation:
```bash
$ go get github.com/functionaloptionstest
```
Run:
```bash
$ cd $GOPATH/src/github.com/evanlinjin/functionaloptionstest
$ go run main.go
```
Help:
```bash
$ go run main.go help
```
